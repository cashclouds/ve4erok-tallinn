# Как добавить фоновое изображение в Hero-блок

У вас есть отличное изображение с силуэтом и дверью со светом. Вот как его добавить на сайт.

## Шаг 1: Сохраните изображение

1. Сохраните ваше изображение (43c49e8b-4e23-4eda-bf43-d47a24f55425.png) в папку с сайтом
2. Переименуйте его в простое имя, например: `hero-background.jpg` или `door-light.jpg`

**Рекомендуемое имя:** `hero-bg.jpg`

## Шаг 2: Оптимизируйте изображение (опционально, но рекомендуется)

Для быстрой загрузки на мобильных устройствах:

1. Откройте изображение в любом фоторедакторе (Paint, Photoshop, онлайн-редактор)
2. Измените размер до ширины 1920px (высота пропорционально)
3. Сохраните с качеством 80-85%

Это сделает сайт быстрее, особенно на мобильных устройствах.

## Шаг 3: Добавьте изображение в CSS

Откройте файл `styles.css` и найдите секцию `.hero` (примерно строка 30).

### Текущий код:

```css
.hero {
    position: relative;
    min-height: 100vh;
    display: flex;
    align-items: center;
    justify-content: center;
    background: linear-gradient(135deg, #0D1117 0%, #1a2332 100%);
    background-image: url('data:image/svg+xml,...');
    background-size: cover;
    background-position: center;
}
```

### Замените на:

```css
.hero {
    position: relative;
    min-height: 100vh;
    display: flex;
    align-items: center;
    justify-content: center;
    background: #0D1117;
    background-image: url('hero-bg.jpg');  /* Ваше изображение */
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}
```

## Шаг 4: Настройте overlay (наложение)

Возможно, вам нужно будет сделать текст более читаемым поверх изображения.

Найдите секцию `.hero-overlay` и попробуйте разные варианты:

### Вариант 1: Темное наложение (сильное)

```css
.hero-overlay {
    background: linear-gradient(to bottom, rgba(13, 17, 23, 0.7), rgba(13, 17, 23, 0.9));
}
```

### Вариант 2: Очень легкое наложение

```css
.hero-overlay {
    background: linear-gradient(to bottom, rgba(13, 17, 23, 0.3), rgba(13, 17, 23, 0.5));
}
```

### Вариант 3: Без наложения (если изображение достаточно темное)

```css
.hero-overlay {
    background: none;
}
```

## Шаг 5: Проверьте результат

1. Откройте `index.html` в браузере
2. Обновите страницу (Ctrl+F5 или Cmd+Shift+R)
3. Проверьте, как выглядит на разных размерах экрана:
   - Компьютер (разверните окно браузера)
   - Планшет (сузьте окно до ~768px)
   - Телефон (сузьте окно до ~375px)

## Альтернатива: Использовать изображение только на больших экранах

Если хотите показывать изображение только на компьютерах, а на телефонах оставить градиент:

```css
.hero {
    background: linear-gradient(135deg, #0D1117 0%, #1a2332 100%);
}

/* Изображение только на экранах шире 768px */
@media (min-width: 768px) {
    .hero {
        background-image: url('hero-bg.jpg');
        background-size: cover;
        background-position: center;
    }
}
```

## Советы по выбору изображения

✅ **Хорошо работает:**
- Темные изображения с контрастными светлыми элементами
- Изображения с естественной композицией (как ваше с дверью)
- Фото с эмоциональным посылом

❌ **Избегайте:**
- Слишком яркие, пестрые изображения (текст будет плохо читаться)
- Изображения с большим количеством мелких деталей
- Очень большие файлы (более 500 КБ)

## Проблемы и решения

**Проблема:** Изображение не отображается
- Проверьте имя файла (регистр важен!)
- Убедитесь, что файл в той же папке, что и CSS
- Попробуйте полный путь: `url('./hero-bg.jpg')`

**Проблема:** Изображение растянуто/искажено
- Используйте `background-size: cover;` (покрывает весь экран)
- Или `background-size: contain;` (вписывает изображение целиком)

**Проблема:** Текст плохо читается
- Увеличьте затемнение в `.hero-overlay`
- Добавьте text-shadow к тексту:
  ```css
  .hero-title {
      text-shadow: 2px 4px 12px rgba(0, 0, 0, 0.8);
  }
  ```
